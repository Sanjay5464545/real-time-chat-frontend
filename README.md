# Real-Time Chat Application - Frontend

React-based frontend for a real-time chat application with WhatsApp-inspired UI.

## Features

- Real-time messaging interface
- Multiple chat rooms support
- Online users list with avatars
- Typing indicators
- Chat history loading
- WhatsApp-inspired UI design
- Responsive layout

## Tech Stack

- **React** (with Vite) - Frontend framework
- **Socket.io-client** - Real-time communication
- **Custom CSS** - Styling

## Installation

1. Clone the repository
2. Install dependencies:

npm install

3. Create `.env` file:

VITE_BACKEND_URL=http://localhost:5000


4. Start the development server:

npm run dev


Application will run on `http://localhost:5173`

## Components

- **JoinRoom** - Entry screen for username and room
- **ChatRoom** - Main chat interface
- **MessageInput** - Message composition area
- **OnlineUsersList** - Shows active users with avatars
- **TypingIndicator** - Displays typing status

## UI Features

- WhatsApp-style message bubbles
- Green theme (#00a884)
- User avatars with initials
- System notifications
- Smooth animations
